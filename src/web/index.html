1
Sign in to Letterboxd Open letterboxd.com in this browser and log in to your account.
2
Load the Chrome extension In Chrome go to chrome://extensions → enable Developer mode → Load unpacked → select the extension/ folder from this repo. Click the extension icon, press Connect, then Start Sync to import your watchlist, diary and lists.
3
Find your session cookie Open DevTools (F12) → Application → Cookies → letterboxd.com → copy the value of letterboxd.user.CURRENT.
